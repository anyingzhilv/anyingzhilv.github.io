【黑果小兵】【微信首发】macOS Big Sur 11.1 20C69 Installer for CLOVER 5127 and OpenCore 0.6.4 and PE三EFI分区原版镜像

12月15日，苹果发布了Big Sur的11.1正式版，版本号为20C69，新版本同时支持INTEL/ARM平台，未来几年内会渐渐取消对INTEL平台的支持，所以黑苹果的FANS务必要且行且珍惜

macOS Big Sur 将强大实力和优美外观的结合提升到一个崭新的高度。精心雕琢的全新设计，让您能淋漓尽致地感受 Mac 的魅力；Safari 浏览器迎来重大更新，待您饱览；地图 App 和信息 App 满载新功能，任您探索；更透明的隐私权限，保护也更周到。

版本: macOS Big Sur 11.1 20C69

发行日期: 2020年12月14日

macOS Big Sur 11.1 带来了对 AirPods Max 的支持、视频 App 增强和 App Store 中的隐私信息。本更新还包括针对 Mac 的错误修复。

AirPods Max

• 支持新款头戴式耳机 AirPods Max

• 高保真音频可提供饱满的音效

• 自适应均衡器可根据个人的耳罩贴合度实时调整音效

• 主动降噪可隔绝环境噪声

• 通透模式可让您听见周围的声音

• 具备动态头部跟踪技术的空间音频可提供影院般的聆听体验

Apple TV

• 全新 Apple TV+ 标签页，让您轻松发现和观看 Apple Original 节目和电影

• 增强的搜索功能，可让您按类别（如类型）浏览并在键入时查看最近搜索和建议

• 最佳搜索结果提供电影、电视节目、演员表、频道和体育中最相关的匹配项

App Store

• App Store 页面上新增隐私信息部分，其中包括由开发者报告的 App 隐私惯例摘要

• Arcade 游戏中的游戏内图表汇总推荐可玩的新 Arcade 游戏

搭载 M1 的 Mac 上的 iPhone 和 iPad App

• 针对 iPhone 和 iPad App 的全新窗口选项，可让您在横排和竖排方向间切换，或扩展窗口以填充整个屏幕

照片

• Apple ProRAW 照片可在“照片” App 中编辑

Safari 浏览器

• Safari 浏览器中新增 Ecosia 搜索引擎选项

空气质量

• 可通过“地图”和 Siri 获取中国大陆各地区的空气质量

• Siri 可针对美国、英国、德国、印度和墨西哥的特定空气质量等级提供健康建议

本更新还解决了以下问题：

• 从 macOS Catalina 升级后打开含时间码轨道的影片时 QuickTime Player 可能会退出

• 蓝牙连接状态未在“控制中心”中显示

• 通过 Apple Watch 自动解锁 Mac 的稳定性

• MacBook Pro 机型上触控板的滚动速度可能比预期更快

• LG UltraFine 5K 显示器在搭载 M1 的 Mac 上可能错误地以 4K 分辨率显示

某些功能并非在所有地区都可用，也并非适用于所有 Apple 设备。

镜像特点：

  • 三个独立的EFI引导分区，同时支持CLOVER / OpenCore / WEPE引导
  • 同时支持 Intel 及 AMD 双平台安装使用，RYZEN CPU请选择 config_AMD_RYZEN 进行安装
  • 新增配置文件，请通过CLOVER-OPTIONS-CONFIGS选择启动：
    • config_iMacPro20_1.plist
    • config_Macmini8_1.plist
    • config_MBA6_2.plist
    • config_MBA9_1.plist
  • 新增32GB安装镜像，适用于32GB及以上容量的U盘制作使用【12-15-2020更新】
  • PE 可连接网络，可远程协助，集成向日葵 和 AnyDesk
  • 附带exFAT分区，用于装入Windows 10等的iso安装镜像，可直接安装双系统
  • 支持UEFI / MBR引导，基本上做到一盘在手，可以直接引导安装某些不支持UEFI引导方式的机型，比如Dell inspiron 7559等。

CLOVER 分区

容量：200MB

  • 12-17-2020 更新

    • Clover默认配置文件config.plist原则上支持各种机型引导安装；
    • 更新CLOVER到v2.5k r5127 ，支持Big Sur安装使用
    • 修复了Preboot分区被隐藏的问题
    • 驱动更新
      • 更新Lilu v1.5.1
      • 更新WhatEverGreen v1.4.6
      • 更新AppleALC v1.5.6
      • 更新VirtualSMC v1.2.0
      • 更新VoodooPS2Controller v2.2.0
      • 新增RTCMemoryFixup v1.0.8
      • 更新BrcmPatchRAM v2.5.6
      • 更新AirportBrcmFixup v2.1.2，支持DW1820A / DW1560 / DW1830 / BCM94331CD / BCM943602CDP / BCM94360CD 等
      • 更新USBInjectAll v0.7.6 新增了对Z490 / B460等芯片组的支持
      • 更新LucyRTL8125Ethernet.kext v1.0.0，支持z490主板

OC 分区

容量：200MB

  • OpenCore v0.6.4正式版，支持GUI图形引导界面，不喜欢勿喷，请自行替换

  • 默认配置文件支持z300芯片组，其它主板请做替换EFI使用

  • 安装结束后请移除引导参数brcmfx-driver=0以使用博通无线网卡

  • 驱动更新

    • 更新Lilu v1.5.1
    • 更新WhatEverGreen v1.4.6
    • 更新AppleALC v1.5.6
    • 更新VirtualSMC v1.2.0
    • 更新VoodooPS2Controller v2.2.0
    • 新增RTCMemoryFixup v1.0.8
    • 更新BrcmPatchRAM v2.5.5
    • 更新AirportBrcmFixup v2.1.2，支持DW1820A / DW1560 / DW1830 / BCM94331CD / BCM943602CDP / BCM94360CD 等
    • 更新USBInjectAll v0.7.6 新增了对Z490 / B460等芯片组的支持
    • 更新LucyRTL8125Ethernet.kext v1.0.0，支持z490主板

WEPE 分区

容量：800MB

微PE工具箱v2.0维护盘增强版64位20200409 版本特点

  • 此维护盘增强版及合盘，纯净无广告

  • 内置网络驱动，可驱动绝大多数有线/无线网卡

  • 内置针对黑苹果的部分工具，方便安装

  • 内置远程协助工具：包括向日葵 和 AnyDesk

  • 在原版基础上升级了内核，增加了些常用工具，使经典微PE更加实用

  • 更新修改内核集成Office迷你版，增加更新维护工具

  • 在由kcyou修复DISM及内核的微PE2.0版核心上进一步DIY折腾而来
